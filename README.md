# モチベーター・マネー
1年　WEBプログラミングⅠ　アプリ開発課題　（2025進級制作展展示作品）

## メール設定
src/MailSender.java 内の "差出人メールアドレス" "Gmailアドレス" "Gmailアプリパスワード"  
これらの部分は起動前に、自身の情報に書き換えてください。  
Gmail は通常のパスワードではなく「アプリパスワード」を使用してください。

## 起動方法
ターミナル内でのみ実行できるコンソールアプリです。  
eclipse での実行を想定して作成されています。  
リポジトリ内には仮のデータが格納された txt ファイルが置かれています。
